# Service Tests

This package contains behavioral tests for the business service layer.

## Contents

- service tests for applications, documents, communications, reminders, schedules, skills, users, sources, locations, timelines, and audit logs

## Responsibility

Service tests should verify business rules, ownership enforcement, validation paths, side effects such as timeline or audit writes, and failure handling.

## Current State Note

These tests should prefer clear behavioral assertions over scaffolding so they track the documented system workflows closely.
